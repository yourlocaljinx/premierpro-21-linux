# Adobe Premier Pro CC 2021 installer for Linux!

## DISCLAIMER:
**Please use this software only if you have an active Adobe Premier Pro subscription. I'm not responsible for any illegal use of this product.**

## Requirements
- An internet connection
- All **read** and **write** rights on your home folder and the folder of installation
- `git`
- `wine` >=6.1 (Avoid 6.20 to 6.22)
- `gdown` - required to download the Premier Pro components (To install: `pip install gdown`)
- `tar`
- `wget`
- `curl`
- Vulkan capable GPU or APU _(optional)_
- `appmenu-gtk-module` _(optional)_
